# 🥷 Destino Shinobi

> Um quiz interativo inspirado no universo de Naruto que descobre qual personagem você seria com base nas suas escolhas.

---

## 🎯 Sobre o projeto

Este projeto foi desenvolvido como um trabalho em grupo para a disciplina de Programação Web.

A proposta consiste na criação de um sistema web interativo capaz de identificar qual personagem o usuário seria dentro de um universo fictício inspirado em Naruto. A aplicação apresenta um quiz com múltiplas perguntas, onde cada resposta contribui para a pontuação de diferentes personagens.

Ao final, o sistema calcula o resultado com base nas escolhas do usuário e exibe o personagem mais compatível, juntamente com uma descrição e imagem.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com foco em organização de código, modularização e aplicação de conceitos de orientação a objetos no front-end.

## 🚀 Como executar

1. **Baixe ou clone o projeto**

```bash
git clone https://github.com/caioxdev/quiz-naruto.git
```

2. **Abra o arquivo** `index.html` no navegador

> Nenhuma instalação ou dependência necessária — é só abrir e jogar! 🎮
